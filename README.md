## Enchantments (60 total)

### Helmet
- **Gravity Well** - Push entities in the surface's facing direction.
- **Rift Ripper** - Open a rift, teleport, and swap places with nearby entities.
- **Black Hole** - Create a pull-damaging black hole (directable/retractable).
- **Glacial Impasse** - Summon moving ice spikes dealing damage.
- **Divine Smite** - Call a lightning rod that strikes periodically.
- **Volcano** - Mark 3 eruption points that deal constant AoE damage.

### Chestplate
- **Milky Chrysalis** - Elytra flight + rocket boost (5s).
- **Planar Shift** - Teleportation circle moving entities on reactivation.
- **Abyss Omen** - Throw a runic stone to summon void AoE damage.
- **Immolate** - Blast that ignites enemies; burning enemies explode on death.
- **Haunt** - Summon a soul-beast that damages on collision.
- **Extinction** - Summon repeated meteor strikes.

### Leggings
- **Tempest** - Air dash (up to 3 times).
- **Gale** - Wind burst dash that also pulls entities.
- **Judgement** - Long leap with explosive landing.
- **Tsunami** - Kick that sends wave knockbacks.
- **Void Wake** - Drop void spikes while sprinting; detonate later.
- **Hurricane** - Forward lunge with spinning kick + knockback.

### Boots
- **Blink** - Short-range teleport forward.
- **Surf** - Ride a wave that drags entities; cancel anytime.
- **Stone Caller** - Summon stone spikes at target location.
- **Stasis** - Freeze non-living entities, releasing with momentum later.
- **Curb Stomp** - Slam down from a jump, creating a shockwave.
- **Soul-bound** - Fire an ember that returns, damaging on contact.

### Weapons
- **Finesse** - Backflip on hit; lunge-strike on activation.
- **Burst** - Consecutive hits increase attack speed; knockback launch.
- **Volatile** - Explosive crits; launch explosive projectiles.
- **Breakthrough** - Disable shields; delete nearest entity after windup.
- **Cascade** - Launch upward on crit, then slam for bonus damage.
- **Tungsten Core** - Heavy knockback + shockwave strike.
- **Viper** - Bonus damage to debuffed foes; summon poison serpent.
- **Molten** - Extra damage to burning targets + ember release.
- **Comet Strike** - Explosive comets that grow with damage dealt.
- **Gust** - Bonus airborne damage; launch enemies skyward.
- **Kinetic** - Gain speed per hit; consume speed for bonus damage.
- **Meathook** - Chain crit targets together; pull them close.

### Tridents
- **Geyser** - Launch enemies upward.
- **Whirlpool** - Vortex pulling everything downward.
- **Undertow** - Pull yourself toward entities/blocks.
- **Upwell** - Blast knocking everything away.
- **Cavitation** - Pull everything to impact center.
- **Slipstream** - Throw yourself instead of trident; it returns.

## Game Changes
- **Remove Base Enchantments** - All vanilla enchants removed.
- **Overhauled Enchanting**
  - New UI with enchantment selection (no RNG).
  - Requires *Chiseled Bookshelves* with enchanted books.
  - Enchanted Books cannot be applied via anvils.
- **Enchantment Limit** - Items only allow *one enchantment*.
- **Consumable Rebalance** - Potions can be thrown further.
- **Trident Rebalance** - Tridents always have Loyalty III.
- **Cooldowns** - Armor + some weapons (e.g., Gust) have cooldowns.
- **Descriptions** - Enchantments show their effects + applicable items.
